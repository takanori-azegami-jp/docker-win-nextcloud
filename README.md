# docker-win-nextcloud

# 環境

WWindows ＋ Docker の Next Cloud を構築

## 環境

- OS：Windows 10 Pro 21H2
- Rancher Desktop 1.9.1
- Docker version 24.0.2-rd, build e63f5fa

## Docker コマンド

```bash
$ docker-compose up -d --build
```

## ブラウザから URL 接続、id/pass でログイン

```
# URL
http://localhost:8080/
```
